# EB corbos Studio简介

EB corbos Studio由基于python实现的启动脚本开始运行，执行脚本时也可以输入参数，相关参数以及帮助信息请执行：

corbos-studio-launcher --help
1

# 启动EB corbos Studio

输入以下命令启动：

corbos-studio-launcher --target-os <target_os>
1

选择对应的workspace： image:{imgdir}/UsingGUI_SelectWsp.png[UsingGUI_SelectWsp, title="Select workspace"]
